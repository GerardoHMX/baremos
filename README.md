# Baremos Educación Física

Aplicación web para profesores de Educación Física: gestión de pruebas, clases, alumnos y baremos.

## Cómo usar

1. Abre `index.html` en el navegador (o sirve la carpeta con un servidor local).
2. Los datos se cargan desde Google Sheets (hojas publicadas en CSV).
3. La escritura de datos usa Google Apps Script (opcional; ver `google-apps-script.md`).

## Tecnologías

- HTML, JavaScript (vanilla)
- Bootstrap 5 y Tailwind CSS
- Datos: Google Sheets (CSV) + Google Apps Script

## Estructura

- `index.html` — Página principal
- `js/app.js` — Lógica de la aplicación
- `js/data.js` — Carga de datos (CSV y URL de Apps Script)
